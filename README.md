# Applaudo's Code Challenge React Native

## Installation for node_modules
**Use yarn, no npm**
```
yarn install
```

## Run on debug mode
```
react-native run-android
```

## Details of development environment
```
- Node version: 12.2.0
- npm version: 6.9.0
- React native version: 0.61.5

```
